# Café Montaña

## Descripción del Proyecto
Solución digital enfocada en fortalecer la presencia en línea de un emprendimiento dedicado a la producción y comercialización de café.

## Tema Seleccionado
Producción y comercialización de café.

## Herramientas Utilizadas
* HTML5
* CSS3 (Diseño visual y estilos de la interfaz)
* JavaScript (Interactividad y funcionalidades del sitio)
* GitHub (Control de versiones)
* Render (Plataforma de despliegue)

## Funcionalidad Principal
Presentación del catálogo de café, historia del emprendimiento y canales de contacto para la comercialización del producto.
